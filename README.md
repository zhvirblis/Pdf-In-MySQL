# PDF in MySQL

## Installation
```sh
$ npm install
 ```
## Configuration

In config/default.json
```js
 {
	"dbConfig":{
		"host": "localhost", //MySQL address
		"user": "root", // username
		"password": "pass", // password
		"database": "db" // name of database for application
	}
}
 ```
## Run
```sh
$ npm start
 ```
